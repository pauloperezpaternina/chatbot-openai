
# Chatbot-openai




## Run Locally

Clone the project

```bash
  git clone git@github.com:pauloperezpaternina/chatbot-openai.git
```

Go to the project directory

```bash
  cd chatbot-openai
```

Install dependencies

```bash
npm install
```

Run

```bash
  npm run dev
```
